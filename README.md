# Desafio-IBM

 ## Como executar o projeto

<br/>

```bash
# Clonar ou baixar o repositório
git clone git@github.com:LarissaSimoes/Desafio-IBM.git
# Entrar no diretório
cd Desafio-IBM
# Instalar as dependências
mvn install
# Rodar a aplicação
Executar a Classe DesafioibmApplication
```
